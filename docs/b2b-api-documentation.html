API B2B Cetronic Benelux

Documentation pour l'intégration de la création de comptes clients depuis un site partenaire.

Endpoint

POST https://cetronicbenelux.com/b2b/signup

Authentification

Chaque requête doit inclure une clé API secrète dans le paramètre key.

Important : Ne jamais exposer la clé API dans du code JavaScript côté client. Les appels doivent être effectués depuis votre serveur backend.

Paramètres

Paramètre Type Description Exemple
key * string Clé API secrète (fournie séparément)
name * string Nom complet du contact Jean Dupont
company * string Nom de l'entreprise Ma Société SPRL
email * string Adresse email (doit être unique) jean@example.com
password * string Mot de passe (minimum 8 caractères) MotDePasse123!
phone * string Numéro de téléphone +32 2 123 45 67
vat * string Numéro de TVA BE0123456789
street * string Nom de la rue Rue de l'Exemple
street_number * string Numéro de rue 42
zip * string Code postal 1000
city * string Ville Bruxelles
country * string Code pays ISO 3166-1 alpha-3 BEL
language * string Langue préférée fr, nl, en

* = Champ obligatoire

Codes pays (ISO 3166-1 alpha-3)

BEL Belgique
FRA France
NLD Pays-Bas
DEU Allemagne
LUX Luxembourg
GBR Royaume-Uni

Réponses

Succès (HTTP 201)

{
    "success": true,
    "message": "User registered successfully. Account pending validation.",
    "user_id": 123
}

Erreur d'authentification (HTTP 401)

{
    "success": false,
    "error": "Invalid API key"
}

Erreur de validation (HTTP 422)

{
    "success": false,
    "errors": {
        "email": ["The email has already been taken."],
        "password": ["The password field must be at least 8 characters."]
    }
}

Exemple d'intégration

PHP (cURL)

<?php
$data = [
    'key' => 'VOTRE_CLE_API',
    'name' => 'Jean Dupont',
    'company' => 'Ma Société SPRL',
    'email' => 'jean@example.com',
    'password' => 'MotDePasse123!',
    'phone' => '+32 2 123 45 67',
    'vat' => 'BE0123456789',
    'street' => 'Rue de l\'Exemple',
    'street_number' => '42',
    'zip' => '1000',
    'city' => 'Bruxelles',
    'country' => 'BEL',
    'language' => 'fr'
];

$ch = curl_init('https://cetronicbenelux.com/b2b/signup');
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, http_build_query($data));
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

$result = json_decode($response, true);

if ($httpCode === 201) {
    echo "Compte créé avec succès ! ID: " . $result['user_id'];
} else {
    echo "Erreur: " . json_encode($result['errors'] ?? $result['error']);
}

Processus de validation

Note importante : Les comptes créés via cette API sont inactifs par défaut.

Un administrateur Cetronic doit valider manuellement chaque compte avant que le client puisse se connecter sur cetronicbenelux.com.
